# 🚐 デイサービス送迎 AI自動配車システム【小規模特化・完全無料】

**定員15〜30名程度のデイサービス**向けに大幅強化した、Googleスプレッドシート＋GASの送迎自動配車ツールです。

毎朝の配車作業を効率化し、**職員負担軽減**と**事業所運営の改善**を支援します。

**[▶ 最新テンプレートを今すぐコピー](https://docs.google.com/spreadsheets/d/1FbShg6yNHtMNQO4xUto4F-D-RV1aYBVtFurwBTV7_8A/copy)**

---

### ✨ 導入で期待できる効果
- 配車作業時間を**大幅削減**（50〜70%程度）
- ベテラン依存の解消
- 若手スタッフの負担軽減 → 離職防止
- 生産性向上により**処遇改善加算**の上乗せを狙いやすく

### ✅ 主な機能（最新版）

- 利用者マスタ・契約・送迎条件の一元管理
- **マスタ同期機能**（基本マスタから自動展開）
- 曜日契約から**当日出席者名簿の自動生成**
- **往路・復路の自動ルート計算**（時間・車椅子・NG乗客・NGドライバー対応）
- **Google Mapsルートリンク自動生成**（各便を地図で確認可能）
- **LINE通知用文章の自動作成**（ドライバーへの共有が簡単）
- 住所から緯度経度の自動取得
- 運行実績の自動蓄積

---

### 📋 シート構成

| シート名                    | 役割 |
|-----------------------------|------|
| `m_user_basics`             | 利用者基本情報 |
| `t_user_contracts`          | 曜日契約 |
| `t_user_pickup_conditions`  | 送迎条件（時間・車椅子・NG） |
| `m_vehicles`                | 車両マスタ |
| `t_daily_attendance`        | 当日出欠・施設情報 |
| `t_routing_outputs`         | 配車結果（Mapsリンク・LINE文章含む） |
| `t_past_records`            | 運行実績 |

---

## 🚀 使い方

1. テンプレートをコピー
2. Apps Scriptに`main.gs`を貼り付け
3. `t_daily_attendance`シートで運行日と施設緯度経度を設定
4. 「名簿作成」→「往路計算」→「復路計算」を実行
5. 出力シートで**Google Mapsリンク**と**LINE用文章**を確認・利用
